# 原神帮助

## 订阅原神日程表

| 关键词              | 说明                                                    |
| ------------------- | ------------------------------------------------------- |
| 原神日历            | **查看本群订阅服务器原神日历/需保持启用原神日历** |
| 原神日历 on/off     | **订阅/取消订阅指定服务器的原神日历推送**         |
| 原神日历 time 时:分 | 设置原神日历推送时间                                    |
| 原神日历 status     | 查看本群原神日历推送设置                                |
| 原神日历 cardimage  | 切换是否使用cardimage模式发送原神日历图片               |
